Part 1: Quick Start README
Prerequisites

Ensure you have the required packages installed:
DOS

pip install numpy scipy matplotlib pyyaml

Setup Environment Variable

Before running the script, you must set the PYTHONPATH to the current root directory so Python can find your local modules.
Run Command

Open your command prompt in the project root directory and execute the following combined command:
DOS

set PYTHONPATH=.&& py examples/calibration_workflow.py

Note: Check the results/ directory immediately after completion for your output visualization files.
